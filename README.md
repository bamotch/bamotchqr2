# BAMOTCH QR

Générateur de QR codes gratuit pour convertir du texte, des liens, des informations Wi-Fi, des contacts et des images en QR codes.

## Comment utiliser

1. **Choisissez le type de contenu** (texte, URL, Wi-Fi, contact, image)
2. **Entrez vos données** dans le formulaire
3. **Personnalisez** la taille et les couleurs
4. **Générez** le QR code
5. **Téléchargez** en PNG, SVG ou JPG
